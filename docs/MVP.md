#MVP Scope - Phase 1

## Include Features
- User Sign Up and Login
- Role-based Access (Student & Instructor)
- Course Creation and Publishing
- Course Enrollment
- Progress Tracking 
- Email Notification on Enrollment

## Out of Scope
- Payment Gateway
- Live Classes
- Mobile Application
- Certificates
- AI Recommandations

## Assumptions
- Users have Internet Connections
- Web Application only
- Moderate Initial Traffic

## Constraints
- No Paid Tools
- Limited Infrastruture
